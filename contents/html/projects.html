Cars Showroom

Cars Showroom: a dynamic car showroom application featuring full CRUD (Create, Read, Update, Delete) operations. Developed using React, Axios, Express, MongoDB, Bootstrap and CORS integration, this project showcases my proficiency in modern web development and database management.

  • React Frontend: Crafted an intuitive user interface with React and Bootstrap, enabling users to seamlessly browse, add, edit, and remove vehicles from the showroom inventory.
  • Express Backend: Built a robust Express.js backend to handle CRUD operations, ensuring efficient data management and seamless interaction between the frontend and backend components.
  • MongoDB Database: Leveraged MongoDB to store and manage showroom data, facilitating fast and flexible access to vehicle information and supporting CRUD functionality.
  • CORS Integration: Implemented CORS support to enable secure communication between the frontend and backend, enhancing the application's security and accessibility.
